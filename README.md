# Bouncing-Ball-on-Websocket-

# Running the Code
 - [Install Node.js and NPM](https://nodejs.org/en/download/)
 - Clone this to your machine
 - cd BouncingBallonWebsocket
 - `npm install`
 - 'node websocket-server.js'
 - Open 'client.html'
